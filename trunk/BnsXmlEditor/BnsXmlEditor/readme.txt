Программа BnsXmlEditor предназначена для редактирования файла перевода Bns нового формата (с autoId и Alias)

Для работы программы необходим .NET Framework 4.0

FAQ:
Загрузка перевода:
	При открытии программы все управляющие элементы заблокированы, кроме пунктов меню "Открыть" и "Выход"
	При выборе пункта "Открыть" появляется диалоговое окно, в котором необходимо указать путь к файлу с переводом (local.xml)
	После открытия файла все элементы разблокируются и список в левой часи окна заполниться элементами из файла.

Перевод элементов:
	При выборе любого элемента из списка соответствующий Alias оригинальный текст и переведенный отобразятся в полях группы "Текст".
	Возможность редактирования предусмотрена только для перевода, оригинальный текст - только просмотр.

Поиск:
	Для выполнения поиска по файлу необходимо ввести на вкладке "Поиск" необходимую ключевую фразу и выбрать поле для поиска.
	Можно отметить галочкой, хотите ли вы учитывать регистр при поиске и использовать ли регулярные выражения.
	После нажания кнопки "Найти" в списке слева останутся только те элементы, которые удовлетворяют запросу.

Замена:
	Для выполнения замены требуемой подстроки в переводе, необходимо перейти на вкладку "Замена", ввести фразу, которую
	нужно заменить, в верхнее поле и фразу, на которую нужно заменить, в нижнее поле. 
	Можно отметить галочкой, хотите ли вы учитывать ли регистр при поиске и испоьзовать ли регулярные выражения.
	При нажатии на кнопку "Заменить все" будет произведена замена подстроки во всех записях исходного файла. После замены будет
	выведено информационное окно, с количестовом элементов, в которых проводилась замена. 
	Список этих элементов будет выведен в левой части окна. 

	ВНИМАНИЕ!!! Отмена операций не поддерживается! Используйте замену очень аккуратно. Перед выполнением замены рекомендую сохранять
	изменения, сделанные до этого момента.

Сохранение перевода:
	При изменении текста в поле для перевода его сохранение в памяти программы происходит автоматически.
	Для сохранения изменений в файл необходимо выбрать пунки меню "Сохранить" или "Сохранить как".
	При выборе "Сохранить" перевод сохранится в тот файл, который был открыт.
	При выборе "Сохранить как" будет показано диалоговое окно, предлагающее сохранить файл в любое место.
